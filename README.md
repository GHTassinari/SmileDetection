# SmileDetection
Código utilizado para a detecção de sorrisos em imagens na Disciplina de computação gráfica no IFSULDEMINAS campus Machado. O repositório inclui dois códigos para sorriso no code.py e no code2.py. O primeiro código reconhece mais assertivamente os sorrisos, enquanto o segundo apesar de reconhecer mais sorrisos, dá mais falsos positivos.

O código utiliza cascades de rostos e sorrisos, para detectar os sorrisos nas imagens, utilizando o Open CV
